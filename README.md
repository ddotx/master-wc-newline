## Setup
```
npx create-nx-workspace --preset=web-components
```
- Add the Storybook plugin
  ```
  yarn add --dev @nrwl/storybook
  ```
- Generating Storybook Configuration
  ```
  nx g @nrwl/storybook:configuration --tsConfiguration=true
  ```

## Run App
```
npm start
```

## Run Storybook
```
nx storybook
```
